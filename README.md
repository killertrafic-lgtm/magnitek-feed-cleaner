# Magnitek Feed Cleaner

Трансформирует remarketing-фид OpenCart (`magnitek.ua`) в **supplemental-фид** для Google Merchant. Ниша — сварочное оборудование и расходники (B2B).

## Что делает
- **google_product_category** (которой нет ни у кого): two-anchor детект — `1238` (аппараты/горелки/плазморезы), `499947` (расходники/материалы), `499927` (маски/защита). Тип детали первым словом → расходник; иначе оборудование.
- **title**: чистка двойного экранирования (`&quot;`→`"`), латинизация процессов (ММА→MMA), нормализация размеров (4,0→4.0, х→×), убрать хвостовой артикул.
- **description**: 80% rich-описаний из фида только чистим; ~20% огрызков → генерим B2B-паспорт (процессы, ток, применение, совместимость).
- **identifier_exists=no + brand=MAGNITEK** всем (тег отсутствует в фиде; своя ТМ без GTIN).
- **product_type** (Сварка > Аппараты/Расходники/Защита/Газовое > подтип) + **custom_label_0..3** (тип / ценовой тир / процесс / совместимость-кластер) для PMax.
- **Валюту НЕ трогаем** — UAH в фиде корректна (отличие от Vparfum).

Бренд MAGNITEK собственный → чужой ТМ нет; совместимость (WP-17/РТ31/ER5356) в title легитимна.

## Авто-обновление
GitHub Actions ежечасно: фид → `transform.py` → `docs/magnitek-supplemental.csv` → коммит при изменении. Safety: <550 товаров → стоп; дорогой товар в дефолт-категории → лог на проверку.
